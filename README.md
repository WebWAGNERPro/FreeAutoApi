## Description

FreeAutoApi est une API REST construite avec **ASP.NET Core** qui permet de gérer des données de manière simple. L'API est connectée à une base de données MySQL et utilise **Swagger** pour générer une documentation interactive des endpoints.

## Prérequis

Avant d'exécuter ce projet, vous devez avoir les outils suivants installés sur votre machine :

- [.NET SDK 6.0 ou supérieur](https://dotnet.microsoft.com/download)
- [MySQL Server](https://dev.mysql.com/downloads/installer/)
- [Docker (si vous utilisez Docker pour MySQL)](https://www.docker.com/products/docker-desktop)
- [Postman ou cURL pour tester les API](https://www.postman.com/)
